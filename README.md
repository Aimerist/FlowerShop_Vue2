# FlowerShop 電商網站 - [Demo](https://aimer-ist.github.io/FlowerShop_VueCLI3/dist/#/)

![FlowerShop](https://i.imgur.com/OsYqkVl.jpg)

## 簡介

此作品  Vue2 、 Vuex 及 Vue Router 製作的購物網站，主要功能有以下：

- 前台:
	1. 電商展示 
       - 商品分類
       - 單一商品介紹
       - 相似商品陳列
       - 動態效果 (視差滾動、輪播效果)
	2. 購物車
	3. 最愛收藏
	4. 優惠卷
	5. 下單/結帳 (表單驗證)
- 後台:
	1. 管理者登入 (身份驗證)
	2. 商品管理
	3. 客戶訂單管理
	4. 優惠卷管理
